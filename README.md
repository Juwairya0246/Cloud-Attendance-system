# Cloud Attendance System ☁️

A simple cloud-based Attendance Management System built using **HTML, CSS, JavaScript, and Firebase Firestore** as part of the UneeQ Cloud Computing Internship.

---

## Features

* Add attendance records (Name, Role, Date, Status)
* Update attendance status (Present / Absent / Late)
* Delete records
* Real-time cloud database sync using Firebase Firestore
* Live dashboard statistics
* Fully responsive UI

---

## Tech Stack

* HTML5
* CSS3
* JavaScript (Vanilla)
* Firebase Firestore (Cloud Database)

---

## Cloud Functionality

This project uses **Firebase Firestore** to store and sync data in real time.
All attendance records are saved in the cloud and automatically updated across sessions.

---

## Features Preview

* Cloud-synced attendance tracking
* Live updates without refresh
* Clean dashboard UI

---

## How to Run

1. Clone or download the repository
2. Open `index.html` in any browser
3. Make sure Firebase config is added correctly
4. Start adding attendance records
